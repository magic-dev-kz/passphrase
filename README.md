# 🔐 PassPhrase

Strong passwords you can actually remember.

**[Try it →](https://magic-dev-kz.github.io/passphrase/)**

## Features
- Three generation modes: Passphrase, Classic random, and PIN
- Real-time strength meter
- Custom word lists (English & Russian included)
- Auto-clear clipboard after copy for security
- Dark / light mode

## Why PassPhrase?
Most password generators spit out strings nobody can memorize. PassPhrase creates strong, human-friendly passphrases from word lists — while still offering classic random and PIN modes when you need them. No cloud, no sync, no risk.

## Tech
- HTML + CSS + JS — no build step
- Works offline (PWA)
- No accounts, no tracking
- Your data stays in your browser (localStorage)
- Free forever
