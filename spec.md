# PassPhrase — генератор запоминаемых паролей для фрилансеров

**Продукт:** веб-приложение (HTML/CSS/JS, без бэкенда)
**Автор спеки:** Саня, продакт-менеджер OpenClaw
**Дата:** 2026-03-29

---

## 1. Persona

**Фрилансеры и удаленщики, у которых 15-30 рабочих аккаунтов и ни одного менеджера паролей.**

Портрет:
- Дизайнер, разработчик, копирайтер, SMM-щик — работает на себя или в маленькой команде
- Аккаунты: Figma, GitHub, Notion, Slack, Upwork, Fiverr, банки, биржи, почты, хостинги
- Боль: пароли либо одинаковые, либо записаны в заметках, либо `Company123!`
- Не хочет ставить 1Password/Bitwarden — "это для параноиков" или "платить за пароли?"
- Хочет: открыл вкладку -> получил нормальный пароль -> скопировал -> закрыл
- Бонус: если пароль можно запомнить без менеджера — это победа

**Ключевой инсайт:** фрилансер не будет ставить расширение и платить $3/мес. Но он откроет закладку в браузере, если она красивая и решает задачу за 3 секунды.

---

## 2. Главная функция

**Генерация запоминаемых парольных фраз (passphrases) с визуальной оценкой силы.**

Не `x7$kQ9!mZ` — а `tiger-velvet-ocean-42` или `КофеЛампаОкно77!`.

Пользователь видит:
1. Сгенерированную фразу
2. Визуальный индикатор силы (не полоска, а нечто интереснее — см. "Изюм")
3. Кнопку "Скопировать"
4. Кнопку "Ещё" (новая генерация)

Всё. Один экран. Три секунды. Задача решена.

## 3. Изюм

### 3.1. Визуальная сила пароля — "Щит"

Вместо скучной полоски "слабый / средний / сильный" — анимированная иконка щита, которая собирается по частям:

| Уровень | Бит энтропии | Визуал | Подпись |
|---------|-------------|--------|---------|
| 1 | < 30 | Разбитый щит, красный | "Взломают за минуту" |
| 2 | 30-50 | Треснутый щит, оранжевый | "Хакер справится за день" |
| 3 | 50-70 | Целый щит, жёлтый | "Нужен серьёзный ботнет" |
| 4 | 70-90 | Щит + меч, зелёный | "Крепость" |
| 5 | > 90 | Щит + меч + корона, фиолетовый с glow | "Легенда" |

Переходы между уровнями анимированы (CSS transitions). Под щитом — конкретная оценка: "~2 года brute-force" (расчёт через zxcvbn или собственная логика).

### 3.2. Режимы генерации

- **Фраза** (по умолчанию): 3-5 случайных слов + разделитель + число/символ. Пример: `hawk-marble-sunset-91`
- **Классика**: случайные символы, длина на слайдере (12-64). Для тех, кто привык
- **PIN**: 4-8 цифр. Для быстрых случаев

Переключение — табы, без перезагрузки.

### 3.3. "Свой вкус" — настройки фразы

Только для режима "Фраза":
- Количество слов: 3 / 4 / 5
- Разделитель: `-` / `.` / `_` / без
- Добавить число: да / нет
- Добавить спецсимвол: да / нет
- Язык слов: English / Русский (два словаря)

### 3.4. Копирование в один клик

Кнопка "Скопировать" с микро-анимацией: иконка -> галочка -> обратно. Работает через `navigator.clipboard`. Фидбек мгновенный, без alert/toast.

### 3.5. История паролей (localStorage)

- Последние 25 сгенерированных паролей хранятся в localStorage
- Выдвижная панель "История" сбоку
- Каждый элемент: пароль (скрыт по умолчанию, показать по клику), дата, кнопка копирования
- Кнопка "Очистить историю" с подтверждением
- Предупреждение: "Пароли хранятся только в этом браузере"

### 3.6. Тёмная / светлая тема

- По умолчанию — по системным настройкам (`prefers-color-scheme`)
- Переключатель в углу
- Выбор сохраняется в localStorage

### 3.7. Анимация генерации

При нажатии "Сгенерировать" — символы пароля "набираются" посимвольно за 300-500ms (эффект печатной машинки). Не блокирует UX, чисто визуальный вкус.

## 4. Acceptance Criteria

### Функциональные

- [ ] **AC-1:** При открытии страницы автоматически генерируется passphrase из 4 слов (английских) с дефисом и числом
- [ ] **AC-2:** Переключение между режимами "Фраза / Классика / PIN" работает мгновенно, без перезагрузки
- [ ] **AC-3:** Настройки фразы (кол-во слов, разделитель, число, спецсимвол, язык) применяются при следующей генерации
- [ ] **AC-4:** Щит силы пароля обновляется в реальном времени при каждой генерации, показывая уровень 1-5 с анимацией перехода
- [ ] **AC-5:** Под щитом отображается оценка времени взлома в понятных единицах ("3 часа", "200 лет", "больше, чем возраст Вселенной")
- [ ] **AC-6:** Кнопка "Скопировать" копирует пароль в буфер обмена и показывает анимацию подтверждения (галочка)
- [ ] **AC-7:** История хранит последние 25 паролей в localStorage, каждый можно показать/скрыть и скопировать
- [ ] **AC-8:** Тёмная/светлая тема переключается и запоминается между сессиями

### Нефункциональные

- [ ] **AC-9:** Страница загружается < 1 секунды. Без фреймворков, без внешних зависимостей (один HTML файл или HTML+CSS+JS)
- [ ] **AC-10:** Работает на мобильных (responsive). Все элементы управления доступны и удобны на экране 375px

## 5. Чего НЕ делать

| Исключено | Почему |
|-----------|--------|
| Бэкенд / база данных | Всё работает на клиенте. Ноль серверов = ноль рисков утечки |
| Регистрация / аккаунты | Фрилансер не будет регистрироваться ради генератора паролей |
| Интеграция с Have I Been Pwned | Требует API-ключ или внешних запросов — усложняет без явной пользы для MVP |
| Менеджер паролей (сохранение + привязка к сайтам) | Это другой продукт. Мы — генератор, не хранилище |
| Экспорт паролей в файл | Создаёт ложное чувство безопасности (файл с паролями на рабочем столе) |
| PWA / Service Worker | Оверинжиниринг для MVP. Добавим если будет трафик |
| Аналитика / трекеры | Генератор паролей с Google Analytics — это анти-маркетинг |
| Мультиязычный UI | Интерфейс на английском. Русские слова — только в словаре для генерации фраз |

## 6. Стек

- **HTML** — семантическая разметка, один файл `index.html` (или split на html/css/js)
- **CSS** — кастомные свойства для тем, анимации на CSS transitions/keyframes, без препроцессоров
- **JS** — vanilla, ES6+, `crypto.getRandomValues()` для генерации (не `Math.random()`)
- **Словари** — встроенные массивы: ~2000 английских слов, ~1000 русских слов (существительные, прилагательные)
- **Шрифт** — системный стек (`-apple-system, system-ui, ...`) для скорости

## 8. Метрика успеха

Продукт хорош, если случайный фрилансер:
1. Открыл страницу
2. Увидел красивый пароль + щит
3. Нажал "Скопировать"
4. Добавил в закладки

Весь путь — меньше 10 секунд. Без единого вопроса "а что тут нажимать?"
